# WebTech HTML & CSS Practice

This repository contains beginner-friendly examples for learning **HTML and CSS**. It includes small demos that demonstrate basic web development concepts such as page structure, forms, images, layouts, positioning, and styling.
The purpose of this project is to practice and understand how web pages are built using HTML and styled using CSS.

## Project Structure

**HTML/**
Contains HTML demos and basic page structure examples:

* Lists and tables (`creationoflist.html`, `creationoftables.html`)
* Forms and input elements (`forms.html`, `exampleforms.html`)
* Hyperlinks (`hyperlinks.html`)
* Media elements (`media.html`, plus `videos/` and `images/` folders)
* HTML attributes and common markup (`attributes.html`)

**css/**
Contains CSS demos and styling examples:

* Display properties (`display.html`, `display.css`)
* Positioning (static, relative, absolute) (`positionproperties.html`, `positionproperties.css`, `postion demo/`)
* Shapes and layout (`shapes.html`, `shapes.css`)
* Text, font, and border styling (`text_font_border_properties.html`, `text_font_border_properties.css`)
* Box model (`css/images/boxmodel.html`, `css/images/boxmodel.css`)
* Gallery and image layouts (`css/gallery/`, `css/images/`)
* A small webpage template (`css/webpage/index.html`, `css/webpage/index.css`)
* Inline, internal, and external CSS usage (`inlinecss/`, `internal css/`, `external css/`)

## How to Preview the Examples

1. Open the project folder.
2. Navigate to any `.html` file.
3. Double-click the file or right-click and open it with a web browser (Chrome, Edge, Firefox, etc.).

The browser will render the HTML page along with its CSS styling so you can see how each concept works.
